# NexusAdmin Route Planner

Aplicação de página única estática para estimativa de rotas com cálculo de frete considerando pedágios e combustível.

## Como utilizar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Informe origem, destino, distância e escolha a tipologia do veículo.
3. Ajuste valores de pedágio, combustível e tarifa por km caso necessário.
4. Clique em **Calcular Frete** para visualizar o resumo do custo e localizar os pontos no mapa.

## Recursos

- Interface inspirada em painéis logísticos.
- Sugestões automáticas de consumo e tarifas por tipologia.
- Mapa interativo com geocodificação via OpenStreetMap.
- Resumo de custos com detalhamento de tarifa base, combustível e pedágios.
